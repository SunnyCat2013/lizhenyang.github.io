# sigmoid function
Sigmoid function 是一个数学函数，数学曲线是 "S" 型，或者说是 sigmoid 弯曲。
通常情况下 sigmoid 函数被认为是 logistic 函数的一个特例。

# Logistic 函数
Logistic 函数是一组函数集合，公式如下：
$$
f(x) = \frac{L}{1 + e^{-k(x - x_0)}}
$$
