969. Pancake Sorting

习惯了 O(n^2) -> O(n)，突然不知道该怎么弄这么指明就要多于 O(n) 的问题了。。
放开限制，表面上想象空间很大，却突然茫然不知所措。

# 一种异类的冒泡

```
class Solution(object):
    def pancakeSort(self, A):
        """
        :type A: List[int]
        :rtype: List[int]
        """
        n = len(A)
        if n < 2:
            return []

        res = []
        for v in range(n, 0, -1):
            idx = A.index(v)
            res.append(idx + 1)
            A = A[:idx + 1][::-1] + A[idx + 1:]
            res.append(v)
            A = A[:v][::-1] + A[v:]
        return res
```
