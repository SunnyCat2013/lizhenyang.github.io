# 796. Rotate String
这个题，可以用到很多知识。对于字符串操作的学习来说，是一个非常好的题。

- 暴力法
-



# 1. 暴力 O(n^2)

```
class Solution(object):
    def rotateString(self, A, B):
        """
        :type A: str
        :type B: str
        :rtype: bool
        """
        na = len(A)
        nb = len(B)

        if na != nb:
            return False

        if A == B: # A and B is an empty string.
            return True

        for i in range(na):
            nstr = A[i:] + A[:i]
            if B == nstr:
                return True

        return False
```
