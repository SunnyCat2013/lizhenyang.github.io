# 903. Valid Permutations for DI Sequence

今天这个题的解法，对以后做 DP 的题来说，很有记性意义。

```
class Solution(object):
    def numPermsDISequence(self, S):
        """
        :type S: str
        :rtype: int
        """
        n = len(S)
        table = [1] * (n + 1)
        for s in S:
            nt = len(table)
            if s == 'I':
                for i in range(1, nt):
                    table[i] += table[i - 1]
                table = table[:nt - 1]
            else:
                for i in range(nt - 1, 0, -1):
                    table[i - 1] += table[i]
                table = table[1:]

        res =  table[-1] if S[-1] == 'I' else table[0]
        return res % (10**9 + 7)


```
